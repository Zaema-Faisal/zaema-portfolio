# Zaema Portfolio Website

A modern and responsive personal portfolio website created using **HTML, CSS, Bootstrap, and Bootstrap Icons**. This portfolio presents personal information, interests, projects, CV access, and contact links in a clean dark-themed design.

---

## GitHub Description

A responsive personal portfolio website built with HTML, CSS, Bootstrap, project cards, CV section, and contact page.

---

## Project Overview

This project is a personal portfolio website for **Zaema Faisal**. It is designed to showcase profile information, interests, projects, resume/CV, and social media links in a professional and user-friendly layout.

The website includes multiple pages such as a home page, projects page, CV page, and contact page. It uses Bootstrap for responsive layout and styling, along with custom CSS for a dark modern theme.

This project was created to practice front-end web development, responsive design, Bootstrap components, navigation bars, project cards, and portfolio presentation.

## Important File Rename Note

Before uploading or running the project, make sure your files are named correctly:

```text
index-html      → index.html
CV-html         → cv.html
projects.hml    → projects.html
```

This is important because the navigation links in the website use:

```text
index.html
projects.html
cv.html
contact.html
```

If the file names are not corrected, some pages may not open properly.

## Required Assets

Zaema_Faisal_CV.pdf
IMG_20250723_134447.jpg

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Make sure all file names are correct.
4. Open `index.html` in your browser.

No installation is required because this is a static front-end website.
